## 1. 项目概览

**ActivityPulse** 是一款 Windows 桌面端电脑行为记录与数据分析工具，基于 PyQt5 构建。

### 核心功能

| 功能 | 说明 |
|------|------|
| 应用使用追踪 | 自动记录前台应用切换，统计使用时长 |
| 输入活动检测 | 监测键盘/鼠标活动，识别活跃与空闲时段 |
| 屏幕截图 | 可配置间隔自动截图，按日期浏览 |
| 仪表盘 | 今日概览：专注时长、活跃应用数、目标进度 |
| 时间轴 | 可视化全天活动轨迹，分钟级精度 |
| 统计洞察 | 环形图/折线图/柱状图多维分析 |
| 分类管理 | 自动推断应用分类，自定义分类与敏感标记 |
| 番茄钟 | 浮动计时器，专注/休息循环 |
| 应用限制 | 设置每日使用时间上限 |
| 久坐提醒 | 长时间未活动弹出提醒 |
| 异常告警 | 检测深夜使用、过度使用等模式 |
| PDF 报告 | 导出每日/每周活动报告 |
| 备份恢复 | 一键打包/恢复数据 |
| 明暗主题 | Light/Dark 双主题 |

### 技术栈

- **Python 3.8+**
- **PyQt5 5.15+** — GUI 框架
- **SQLite** — 本地数据存储（WAL 模式）
- **pynput** — 键鼠监听
- **Pillow** — 截图处理
- **psutil** — 进程信息
- **pygetwindow** — 窗口标题获取
- **PyInstaller** — 打包为可执行文件

## 4. 核心模块详解

### 4.1 MainWindow（主窗口）

**文件**: `gui/main_window.py`

主窗口是应用的编排中心，负责：

- 初始化 `DatabaseManager` 和三个监控器
- 构建侧边栏 + 页面堆栈（QStackedWidget）
- 管理主题切换、隐私模式、监控状态
- 注册全局快捷键
- 初始化系统托盘

```python
class MainWindow(QMainWindow):
    def __init__(self):
        self.db = DatabaseManager()       # 数据库
        self._init_monitors()             # 监控器
        self._setup_ui()                  # UI 构建
        self._init_managers()             # 托盘/检查/导出管理器
        self._init_post_setup()           # 后续设置
```

### 4.2 DatabaseManager（数据库管理器）

**文件**: `database/db_manager.py`

采用 **Mixin 模式** 组织数据库操作，通过多继承组合功能：

```python
class DatabaseManager(
    AppUsageMixin,      # 应用使用记录
    ScreenshotMixin,    # 截图记录
    SettingsMixin,      # 配置项
    IdleMixin,          # 空闲时段
    ActivityTagMixin,   # 活动标签
    AppLimitMixin,      # 应用限制
    AnomalyMixin,       # 异常告警
):
```

**线程安全机制**：
- 全局写锁 `self._lock = threading.Lock()`
- 线程本地连接 `self._local = threading.local()`
- WAL 模式 `PRAGMA journal_mode=WAL`
- 配置缓存 `self._config_cache`

### 4.3 监控器

三个监控器均运行在独立线程中：

| 监控器 | 类 | 间隔 | 说明 |
|--------|-----|------|------|
| 应用监控 | `AppMonitor` | 5s（可配） | 追踪前台窗口应用名+标题 |
| 输入监控 | `InputMonitor` | 2s（可配） | 键鼠事件缓冲写入 |
| 截图监控 | `ScreenMonitor` | 60s（可配） | Pillow 截图保存 |

所有监控器通过 `start()`/`stop()` 控制，数据写入 `DatabaseManager`。

## 7. 对话框基类模式

**文件**: `gui/components/base_dialog.py`

所有对话框统一继承 `BaseDialog`，消除重复的主题管理代码：

```python
class BaseDialog(QDialog):
    def __init__(self, is_dark=False, parent=None, dialog_style="dialog_base"):
        super().__init__(parent)
        self._is_dark = is_dark
        self._colors = get_colors(is_dark)
        self._dialog_style = dialog_style
        if dialog_style:
            self._apply_dialog_style()

    def set_theme(self, is_dark):
        self._is_dark = is_dark
        self._colors = get_colors(is_dark)
        if self._dialog_style:
            self._apply_dialog_style()
```

**dialog_style 参数**：
- `"dialog_base"` — 基础对话框样式
- `"dialog_card"` — 卡片式对话框样式
- `""` — 不自动应用样式（子类自行管理）

**继承示例**：
```python
class MyDialog(BaseDialog):
    def __init__(self, parent=None):
        super().__init__(is_dark=False, parent=parent, dialog_style="dialog_card")
        self._setup_ui()

    def set_theme(self, is_dark):
        super().set_theme(is_dark)  # 更新 _colors + 背景样式
        # ... 更新子控件样式
```

## 9. 监控器

### 9.1 AppMonitor

```python
monitor = AppMonitor(db_manager, interval=5.0)
monitor.start()   # 启动后台线程
monitor.stop()    # 停止并记录最后一条
```

- 每 `interval` 秒获取前台窗口（pygetwindow + psutil 备选）
- 应用切换时记录上一条使用时长
- 隐私模式暂停记录

### 9.2 InputMonitor

```python
monitor = InputMonitor(db_manager, save_interval=2.0, track_mouse_move=False)
monitor.start()
monitor.stop()
```

- pynput 监听键盘按键 + 鼠标点击/滚动
- 事件缓冲，每 `save_interval` 秒批量写入
- 不记录具体按键内容（隐私保护）

### 9.3 ScreenMonitor

```python
monitor = ScreenMonitor(db_manager, interval=60.0, quality=80, save_screenshots=True)
monitor.start()
monitor.stop()
```

- Pillow ImageGrab 定时截图
- 自动生成缩略图
- 截图保存至 `~/.computer_monitor/screenshots/YYYY-MM-DD/`

## 11. 开发规范

### 代码结构约定

1. **对话框**：继承 `BaseDialog`，使用 `dialog_style` 参数
2. **QSS 样式**：优先使用 `QSS_STYLES[key].format(c=colors)`，避免内联样式
3. **颜色引用**：使用 `get_colors(is_dark)` 返回的字典，禁止硬编码颜色值
4. **主题切换**：实现 `set_theme(is_dark)` 方法，更新 `self._colors` 缓存
5. **数据库操作**：通过 `DatabaseManager` 的 Mixin 方法，不直接执行 SQL
6. **线程安全**：数据库写操作通过 `self._lock` 保护

### 长函数拆分策略

- ≥50 行的函数考虑拆分
- 按逻辑块提取私有子方法（`_` 前缀）
- 主方法变为编排调用
- 保持原有行为不变

## 12. 常见问题

### Q: PyInstaller 打包报 "Qt plugin directory does not exist"

**A**: 项目路径含中文字符导致。创建 junction link 到纯英文路径后打包。

### Q: 运行时缺少依赖

**A**: 执行 `pip install -r requirements.txt`，或查看启动时的错误提示。

### Q: 数据库锁定 (SQLITE_BUSY)

**A**: DatabaseManager 使用 WAL 模式 + 线程锁，正常情况不会出现。如出现，检查是否有外部进程访问数据库文件。

### Q: 截图功能不工作

**A**: 确认 Pillow 已安装，且 `screenshot_enabled` 配置为 `"1"`。

### Q: 全局快捷键不生效

**A**: pynput 需要管理员权限监听全局键盘事件。以管理员身份运行可解决。
