# 电脑行为记录软件 - 桌面UI重构方案

> 基于《初步.docx》和《进阶.docx》两份文档，适配 PyQt5 桌面端实现

---

## 一、整体架构

### 布局模式：经典侧边栏模式
```
┌─────────────────────────────────────────────────┐
│  ┌──────────┐  ┌──────────────────────────────┐  │
│  │          │  │  顶部工具栏                    │  │
│  │  侧边栏  │  │  (时间选择器 + 导出按钮)       │  │
│  │  导航     │  ├──────────────────────────────┤  │
│  │          │  │                              │  │
│  │  ·仪表盘 │  │                              │  │
│  │  ·时间轴 │  │     右侧内容区               │  │
│  │  ·统计   │  │     (卡片式布局)             │  │
│  │  ·分类   │  │                              │  │
│  │          │  │                              │  │
│  │ ──────── │  ├──────────────────────────────┤  │
│  │ 🟢 记录中│  │  隐私保护提示栏               │  │
│  │ ⏸ 暂停   │  └──────────────────────────────┘  │
│  │ ⚙ 设置   │                                    │
│  └──────────┘                                    │
└─────────────────────────────────────────────────┘
```

- 默认窗口尺寸：1100 x 700，最小 900 x 600
- 侧边栏宽度：220px（展开）/ 60px（折叠仅图标）
- 卡片间距：16px，圆角：8px

---

## 二、侧边栏导航 (Sidebar)

### 品牌区
- Logo图标 + 软件名称 "行为记录"
- 点击可折叠为纯图标模式

### 核心导航（4个页面）
| 图标 | 名称 | 说明 |
|------|------|------|
| 📊 | 仪表盘 | 全局数据概览，核心指标+热力图+Top5 |
| 🕐 | 时间轴 | 甘特图样式行为流，语义化配色 |
| 📈 | 统计洞察 | 环形图/趋势图/柱状图深度分析 |
| 🏷️ | 分类管理 | 自定义应用标签，敏感应用脱敏 |

### 底部操作区
- **状态指示灯**：绿色呼吸灯 = 记录中，红色 = 已暂停
- **快捷开关**：暂停记录 / 隐私模式 切换按钮
- **设置入口**：齿轮图标，打开设置页

## 三、各页面详细设计

### 页面1：仪表盘 (Dashboard)

```
┌─────────────────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│  │ 专注时长  │ │ 活跃应用  │ │ 操作次数  │     │
│  │ 4h 20m   │ │   12     │ │  1,247   │     │
│  │ 较昨日+15%│ │ 较昨日-2 │ │ 较昨日+89│     │
│  └──────────┘ └──────────┘ └──────────┘     │
│                                              │
│  ┌──────────────────────────────────────┐    │
│  │  活跃热力图 (类GitHub贡献图)          │    │
│  │  ████░░████░░░░████████░░░░░░████    │    │
│  │  (24小时 x 7天 矩阵色块)             │    │
│  └──────────────────────────────────────┘    │
│                                              │
│  ┌──────────────────────────────────────┐    │
│  │  今日 Top 5 应用                      │    │
│  │  🖥 VS Code    ████████████  2h 30m  │    │
│  │  🌐 Chrome     ████████      1h 45m  │    │
│  │  💬 WeChat     ████          0h 40m  │    │
│  │  📝 Notepad    ██            0h 15m  │    │
│  │  🎮 Steam      █             0h 10m  │    │
│  └──────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

**核心要素：**
- 3个顶部指标卡：专注时长、活跃应用数、操作次数，含环比变化
- 活跃热力图：24列(小时) x 7行(天) 矩阵，颜色深浅表示活跃度
- Top 5列表：应用图标 + 名称 + 进度条 + 时长

- 悬停Tooltip显示详情
- 点击色块可展开编辑面板（拆分/合并/重新分类）

## 六、隐私与信任感 UI

1. **隐私模式**：开启后界面叠加半透明红色遮罩 + 顶部红色提示条
2. **本地存储标识**：仪表盘底部盾牌图标 + "所有数据均存储于本地"
3. **敏感应用脱敏**：分类管理中可标记敏感应用，图表中自动隐藏名称

## 七、视觉风格

### 色彩系统
| 用途 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 主背景 | #F9FAFB | #111827 |
| 卡片背景 | #FFFFFF | #1F2937 |

### 字体
- 标题/正文：Microsoft YaHei
- 数据/时间：Consolas（等宽字体，数字对齐）

### 微交互
- 按钮 Hover：轻微上浮 + 阴影加深
- 页面切换：150ms 淡入淡出
- 数据加载：骨架屏占位

## 八、技术实现要点（PyQt5适配）

| 文档建议 | PyQt5实现方案 |
|---------|--------------|
| 侧边栏导航 | QFrame + QPushButton 自定义样式 |
| 卡片布局 | QFrame 带圆角阴影 + QVBoxLayout |
| 热力图 | QPainter 自绘矩阵色块 |
| 甘特图时间轴 | QPainter 自绘色块 + QScrollArea |
| 环形图 | QPainter 自绘 或 pyqtgraph |
| 折线图/柱状图 | pyqtgraph 轻量图表库 |
| 系统托盘 | QSystemTrayIcon |
| 深色/浅色主题 | QSS样式表切换 |
| 页面切换 | QStackedWidget |

---

## 九、开发优先级

1. **P0 - 框架搭建**：侧边栏导航 + QStackedWidget页面切换 + 主题系统
2. **P1 - 仪表盘**：指标卡 + Top5列表 + 热力图
3. **P2 - 时间轴**：甘特图 + 语义化配色 + Tooltip
4. **P3 - 统计洞察**：环形图 + 折线图 + 柱状图（需pyqtgraph）
5. **P4 - 分类管理**：应用分类 + 敏感标记
6. **P5 - 增强功能**：系统托盘 + 隐私模式 + 导出功能
